Command Line Options

-h
Used to display the help menu
-m
Used for including the 'home icon' in the topnav
-i (NOT IMPLEMENTED)
Used to activate index.html based urls (to browse site without a webserver)
-x
Match template based on the file extension only (for single layout sites)
-o:folder
Used specify an output folder (different than site)
-r:folder
Used specify the root path. Used when your files are not located at the website root
-v
Used to enable verbose mode
-d
Used to enable debug mode

Subtitution tags

{$root}
Used inject the root folder of the site (for styles & scripts)
{$topnav}
Used inject the top navigation menu structure
{$breadcrumb}
Used inject the breadcrumb
{$search}
Used inject the search data in the search page
{$children}
Used inject links of the direct children of the page
{$url}
Current page URL
{$date}
Current date (yyyy/mm/dd)
{$time}
Current time (hh:mm)
{$gmt}
Current GMT datetime (yyyymmddThhmmssZ)
